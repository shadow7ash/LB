## Configuration
- `TELEGRAM_BOT_TOKEN`: Your Telegram bot token.
- `OWNER_ID`: Your Telegram user ID.
- `FORCE_SUBSCRIBE_MESSAGE`: Message to prompt users to subscribe to your channel.
- `MONGODB_URL`: Your MongoDB connection URL.
